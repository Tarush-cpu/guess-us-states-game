# U.S. States Guessing Game 🗺️

A desktop application that tests your knowledge of United States geography. 

## About The Project
This is a geographic puzzle game where users type in the names of U.S. states to populate a blank map. It processes user input, cross-references it with a dataset of state coordinates, and dynamically renders the state name on the map if the guess is correct. 

### Built With
* **Python 3**
* **Turtle Graphics:** Used for rendering the map UI and writing text to the screen.
* **Pandas:** Used for reading and extracting coordinates from the `50_states.csv` dataset.

## How to Run
To run this game locally on your machine:
1. Ensure you have Python installed.
2. Clone this repository to your local machine.
3. Keep `main.py`, `50_states.csv`, and `blank_states_img.gif` in the same directory.
4. Run `main.py` from your terminal.
